Buttplug C# Version 0.1.0 Released!

Our first release for Windows 7/8/10 is out! Downloads are available at our github repo, and a tutorial is available on our forums. We currently support XInput (Xbox) Gamepads, Fleshlight Launch, Lovense, Vibratissimo, and Magic Motion toys. More coming soon!

If you're interest in movie sync, check out Syncydink, our open source Web based haptic movie player, based on Vue.js and typescript (and with WebVR support!). There's also the first community project to support Buttplug, ScriptPlayer, a open source native Windows haptic movie player in C#!

Support for more platforms (macOS, iOS, Linux, Android, ChromeOS) coming soon!
